Python Fundamentals: Guía de Actividades
Este repositorio contiene un Notebook de Jupyter (.ipynb) con una serie de 10 actividades diseñadas para aprender y practicar los conceptos fundamentales de programación en Python.
Las actividades abarcan desde el clásico "Hola Mundo" hasta un desafío integrador que combina lógica condicional y bucles.

🚀 Contenido del Notebook
El archivo actividades.ipynb está organizado de forma progresiva:

Fundamentos Básicos: Manejo de la función print, creación de variables y reconocimiento de tipos de datos (str, int, float, bool).

Interacción con el Usuario: Uso de input() y conversión de tipos para realizar operaciones aritméticas.

Control de Flujo (Condicionales): Implementación de estructuras if, elif y else para toma de decisiones y uso del operador módulo %.

Bucles e Iteraciones: * While: Uso de contadores y acumuladores para cálculos estadísticos.

For: Generación de secuencias con range() (ej. tablas de multiplicar).

Desafío Integrador: Un programa completo que procesa una lista de números para encontrar máximos, mínimos, sumas, promedios y clasificación de pares/impares.
